# ScriptFlow 脚本剪辑器 - 技术规格文档

## 1. 概念与愿景

ScriptFlow 是一款将传统静态脚本写作转变为可视化时间轴剪辑体验的 Web 应用。它让脚本创作者像剪辑视频一样剪辑文字，通过直观的拖拽、拉伸和情绪可视化，找到脚本的最佳节奏结构。核心理念："像剪辑视频一样，剪辑你的脚本。"

## 2. 设计语言

### 美学方向
灵感来源：视频剪辑软件（Premiere/Final Cut）的专业感 + 现代极简工具的清爽。深色主题为主，减少视觉疲劳，突出内容本身。

### 字体系统
- 标题：`"Space Grotesk", "Noto Sans SC", sans-serif` - 科技感强
- 正文：`"Inter", "Noto Sans SC", sans-serif` - 清晰易读
- 代码/时间码：`"JetBrains Mono", monospace`

### 空间系统
- 基础单位：4px
- 间距：4, 8, 12, 16, 24, 32, 48, 64
- 圆角：4px（小元素）, 8px（卡片）, 12px（面板）
- 阴影：仅在悬浮/激活状态使用，柔和发光效果

### 动效哲学
- 所有交互：150-250ms，ease-out
- 拖拽操作：实时反馈，无延迟
- 片段插入：弹性动画（spring effect）
- 波形绘制：平滑过渡动画
- 状态切换：渐变过渡

### 视觉资产
- 图标库：Lucide React（线性风格，2px stroke）
- 装饰：网格背景纹理、微妙渐变
- 轨道：毛玻璃效果

## 4. 功能与交互

### 4.1 智能解析模块
- **文本输入**：大文本框，支持粘贴和直接输入
- **解析按钮**：点击后显示加载动画，调用 NLP API
- **解析结果**：将文本拆分为独立片段，每个片段包含：
  - 原文内容
  - 建议持续时间（基于字数和标点计算）
  - 句子类型标签（陈述/疑问/感叹/停顿）
  - 情绪分数（-1 到 1）
- **逐个编辑**：点击片段可进入编辑模式

### 4.2 时间轴剪辑模块
- **时间刻度**：顶部显示时间轴（分:秒:帧），可缩放
- **片段操作**：
  - 拖拽移动：实时预览插入位置，显示蓝色引导线
  - 拉伸边缘：调整持续时间，显示时间变化
  - 双击：进入文本编辑模式
  - 右键菜单：复制/删除/分割/设为关键片段
- **间隙管理**：片段间可插入"气口"（pause），以灰色方块表示
- **多轨道**：
  - A-Roll 轨道：橙色主题，显示口播文本
  - B-Roll 轨道：蓝色主题，显示画面备注
  - 轨道可独立滚动，可折叠/展开
- **吸附功能**：拖拽时自动吸附到其他片段边缘或时间刻度

### 4.3 情绪节奏可视化
- **波形图**：基于片段情绪分数和时长生成
- **颜色编码**：
  - 高潮（情绪>0.6）：红色峰值
  - 中性（0.2-0.6）：黄色过渡
  - 低沉（<0.2）：绿色波谷
- **交互**：点击波形位置跳转到对应片段
- **可折叠**：用户可隐藏波形区以获得更大编辑空间

### 4.4 导出模块
- **导出 A-Roll 提词器**：
  - 生成带时间码的纯文本
  - 格式：`[00:00:15] 你好，欢迎来到今天的视频...`
- **导出 B-Roll 分镜表**：
  - 生成结构化表格（CSV 或 Markdown）
  - 包含：时间点、画面描述、关键动作、道具、BGM 建议
- **导出编辑标记**：
  - 生成 JSON 格式，包含所有时间点和标签

### 4.5 其他功能
- **模板系统**：预设 5 种模板
  - 知识科普：平稳开头 → 信息密度上升 → 高潮总结
  - 产品测评：痛点引入 → 产品亮点 → 行动号召
  - 故事叙事：悬念开头 → 冲突发展 → 解决高潮
  - 日常 Vlog：轻松开场 → 事件串联 → 感悟结尾
  - 开箱体验：期待感 → 展示 → 评价总结
- **TTS 预览**：集成 Web Speech API，点击播放当前选中的片段或整个时间轴
- **协作功能**：生成分享链接（只读或可编辑）

### 状态处理
- **空状态**：显示引导文字和"粘贴文稿开始"按钮
- **加载状态**：骨架屏 + 脉冲动画
- **错误状态**：红色边框 + 错误提示 + 重试按钮

## 5. 组件清单

### Header
- Logo + 项目标题
- 模板下拉选择器
- 协作按钮（带链接图标）
- 导出下拉菜单
- 状态：default

### TextInputPanel
- 大文本输入框（占底部面板左侧 40%）
- 解析按钮
- 状态：empty / has-content / parsing / parsed

### TimelineHeader
- 时间刻度（支持缩放）
- 当前播放头位置指示器
- 状态：default / zoomed-in / zoomed-out

### ClipBlock
- 片段主体（圆角矩形，显示文字预览）
- 左侧手柄（拖拽）
- 右侧边缘（拉伸）
- 情绪指示条（顶部小色条）
- 状态：default / selected / dragging / resizing / editing

### EmotionWaveform
- SVG 波形区域
- 时间轴对应标记
- 状态：collapsed / expanded

### TrackLane
- 轨道标签（A-Roll / B-Roll）
- 折叠/展开按钮
- 片段容器
- 状态：default / collapsed / receiving-drop

### GapBlock
- 灰色方块代表"气口"
- 可拖拽调整长度
- 状态：default / selected / dragging

### PropertiesPanel
- 片段属性编辑表单
- 时间码输入
- 情绪微调滑块
- 备注输入框
- 状态：no-selection / single-selection / multi-selection

### ExportModal
- 导出选项卡（提词器/分镜表/标记文件）
- 预览区
- 下载按钮
- 状态：closed / open

### ToastNotification
- 成功/警告/错误提示
- 自动消失（3秒）
- 状态：entering / visible / exiting

## 6. 技术架构

### 前端
- **框架**：React 18 + Vite
- **状态管理**：Zustand
- **样式**：Tailwind CSS + CSS Variables
- **拖拽**：@dnd-kit/core
- **时间轴**：自定义实现（基于 HTML5 Drag & Drop）
- **TTS**：Web Speech API
- **图标**：Lucide React

### 后端（Zeabur 部署）
- **运行时**：Node.js 18+
- **框架**：Express.js
- **NLP**：简化的本地分析（正则 + 标点分析）
- **情绪分析**：规则基础分析（检测感叹号、问号、情感词）

### 数据模型
```typescript
interface Project {
  id: string;
  name: string;
  createdAt: Date;
  updatedAt: Date;
  template: TemplateType | null;
  clips: Clip[];
  gaps: Gap[];
  settings: ProjectSettings;
}

interface Clip {
  id: string;
  track: 'a-roll' | 'b-roll';
  content: string;
  startTime: number;    // 秒
  duration: number;     // 秒
  emotion: number;      // -1 到 1
  type: 'statement' | 'question' | 'exclamation' | 'pause';
  isKeyMoment: boolean;
  notes: string;
}

interface Gap {
  id: string;
  track: 'a-roll' | 'b-roll';
  startTime: number;
  duration: number;
}

interface ProjectSettings {
  defaultTtsSpeed: number;
  showEmotionWaveform: boolean;
  snapToGrid: boolean;
}
```

### 存储
- 本地：localStorage 保存项目数据
- 协作：生成分享码，数据存储在 URL 中（Base64 编码）
